### Project 2

# Decode Labs UI/UX Internship – Project 2

## Overview

This project focuses on designing a low-fidelity homepage wireframe for an e-commerce platform. The goal was to create a clean layout with intuitive navigation and a clear shopping journey.

## Problem Statement

Design an e-commerce homepage that enables users to easily discover products, browse categories, and begin their shopping experience.

## User Flow

Home → Browse Categories → Product Details → Add to Cart → Checkout → Payment → Order Confirmation

## My Role

- Information Architecture
- UX Planning
- Low-Fidelity Wireframing

## Tools Used

- Figma

## Key Design Decisions

- Simple navigation bar
- Prominent hero section with CTA
- Product categories
- Best-selling products
- Promotional banner
- Trust indicators
- Newsletter section
- Footer navigation

## Deliverables

- Homepage Wireframe
- User Flow
- UX Documentation
