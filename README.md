# Decode Labs UI/UX Internship Tasks

This repository contains all UI/UX design tasks completed during my internship at Decode Labs.

## About

Throughout this internship, I worked on multiple design assignments to strengthen my skills in user experience design, user interface design, wireframing, visual design, and prototyping.

All projects were designed in **Figma** following a user-centered design process.
